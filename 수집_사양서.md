# 덕플레이스 가격 데이터 수집 사양서 (서울)

> 이 문서는 **별도의 수집(스크래핑) 세션**이 따라야 할 지침이다. 프로젝트 전체 맥락은 `CLAUDE.md`, 기획 원문은 `claude_code_첫메시지.md` 참고.
>
> **✅ 완료 (2026-05-31)** — 이 사양서대로 수집·정제 끝. 결과: `data/seoul/price/덕플레이스_서울_모델용.csv`(2,555 정류장, Y=원/면). 데이터 사전: `data/seoul/README.md`. 이후 X피처·모델링까지 전부 완료(산출물 `model/`). (수집은 robots.txt 차단 때문에 브라우저(Claude in Chrome) 내 `fetch()`로 진행함.)

## 목표
서울 25개 자치구의 덕플레이스(DUK PLACE) 버스정류장 광고 상품을 **전수 수집**해, 정류장별 **광고가(Y) + 매칭용 ARS 번호 + 정규화/필터용 속성**을 한 테이블로 만든다. 예상 규모 **N ≈ 2,500~3,000개 정류장**.

## 사이트 구조 (2026-05 확인)
- 진입: "가격 비교" 페이지 → **자치구 카드** 클릭 → 그 구의 **정류장 목록** → 각 **정류장 상세 페이지**.
- **로그인 불필요. 구별 목록에 페이지네이션 없음**(목록 전체가 한 번에 로드).
- 자치구 카드에 요약(정류장 개수·평균·최저·최고가)도 표시됨.
- 모든 가격은 **부가세·제작비·설치비 포함 실결제가 / 월(1개월) 단가**(사이트 명시).
- 한 정류장은 면 옵션이 여러 개일 수 있음: **단면(1면) / 양면(2면) / 전체(4면)**. (한 패널=4면, 양면=2면=한쪽 면)

## 범위
- **서울 25개 자치구만.** 목록에 "수도권"(경기·인천)이 섞여 있으면 **제외** — 후속 데이터(좌표 마스터·공시지가 등)가 서울 한정이다.
- **영상(디지털) 광고도 수집은 하되** `상품유형`으로 구분(메인 모델에선 제외, 별도 분석 여지). 포맷이 mp4·mov·wmv면 영상.

## 수집 단위 & 컬럼 (정류장 1개 = 1행)
| 컬럼 | 설명 | 비고 |
|---|---|---|
| `자치구` | 예: 서초구 | |
| `정류장이름` | 사이트 표기 원문(끝 `[#####]` 포함) | |
| `ARS` | 이름 끝 대괄호 5자리 | **반드시 텍스트로 저장. 4자리면 앞에 `0` 패딩** |
| `가격_원` | 대표 표시가격(숫자) | 콤마·"원" 제거, 정수 |
| `가격기준` | `단면(1면)` / `양면(2면)` / `전체(4면)` | 그 대표가격이 몇 면 기준인지 |
| `가격_양면` | 양면(2면) 가격 | 상세에 있으면, 없으면 공란 |
| `가격_4면` | 전체(4면) 가격 | 상세에 있으면 |
| `상품유형` | `인쇄` / `영상` | 포맷으로 판별 |
| `지원포맷` | ai,psd / mp4,mov,wmv … | |
| `운영시간` | 예: 24시간 노출 | |
| `야간조명` | O / X | |
| `혼잡도` | 원활/보통/혼잡/매우혼잡 | 덕플 예약지표(모델 피처 아님, 참고) |
| `상세URL` | 정류장 상세 페이지 링크 | 재확인·QC용 |
| `수집일시` | 수집 시각(YYYY-MM-DD) | |

> **모델에 꼭 필요한 것: `ARS`, `가격_원`, `가격기준`, `상품유형`.** 나머지는 정제·필터·참고용. (면 옵션 가격을 같이 받아두면 2단계에서 '원/면'으로 깔끔히 정규화 가능 → 가능하면 `가격_양면`/`가격_4면`도 채울 것.)

## 품질 체크 (수집 직후 스크립트로)
1. `ARS` 텍스트·5자리(leading zero) 보장. **중복 ARS 제거**(같은 정류장 2번 수집 방지).
2. 수집본을 좌표 마스터 `data/seoul/서울시버스정류소위치정보(20260506).xlsx`(키=`ARS_ID`)와 조인 → **매칭률 리포트** 출력, 못 맞은 ARS 목록 표시. (서초 14개 테스트에서 14/14 매칭됨 → 높은 매칭률 기대.)
3. **이름 유사도** 낮은 매칭 플래그(덕플 `정류장이름` vs 마스터 `정류소명`) — ARS 오기입/재배정 의심 건 수동 확인용. (정상 예외: 재건축·개명, 예 `22607` LH서초아파트→호반써밋서초파크.)
4. 가격 이상치(0원·공란·비정상) 점검. 영상상품 개수 집계. 자치구별 개수가 카드 요약과 대략 맞는지 대조.
