# 서울 버스정류장 광고가 예측 도구 (seoul-bus-ad)

도시공학 "데이터기반도시설계" 팀 프로젝트. **좌표를 입력하면 그 위치 버스정류장의 월 광고 단가(원)를 예측**하는 회귀 모델 + 도구를 만든다. 단순 예측이 아니라 "도시의 물리적 구조가 입지의 상업적 가치를 어떻게 결정하는가"를 SHAP으로 해석해 보이는 게 수업적 핵심.

> 전체 기획 원문(단일 진실 소스): `claude_code_첫메시지.md` — 세부는 항상 이 문서를 우선 참고.

## 확정된 핵심 결정
- **대상 도시: 서울** (세종은 폐기. `data/sejong/`는 참고용으로만 보존, 분석에 사용 안 함).
- **문제 유형: 회귀**(연속값 광고가). 분류·정확도% 아님 → 평가는 MAE(주)·RMSE·MAPE·R².
- **Y(광고가) 조건**: 덕플레이스 수집, **월 단가 / 부가세·제작비·설치비 포함 실결제가**. ⚠️ 처음 가정과 달리 **단면(1면)·양면(2면) 기준이 섞여 있음**(샘플 확인) → 2단계에서 **면수 기준 정규화** 필요(예: 원/면, 또는 양면 환산. 패널당 면수는 신규 좌표에서 모르므로 Y를 표준화해 입지가치만 학습). **영상 광고(예: 강남역)는 별도 상품**(디지털·제작비 미포함·면 기준 없음) → 메인 모델에서 제외. 수집 시점=현재.
- **정류장 식별**: 덕플레이스 제목 끝 대괄호 **5자리 = ARS 번호**로 추정 → 이 ARS로 열린데이터광장 좌표와 매칭. **ID는 항상 텍스트로 다뤄 leading zero 보존.**
- **피처(X)는 좌표에서 산출 가능한 도시구조만.** 신규 위치에서 못 구하는 정보(과거 광고이력 등) 금지. 우선순위 1·2·3·5 먼저, 4·6은 여력 시. (덕플레이스의 조명·지원포맷·혼잡도 등 패널/수요 속성은 좌표산출 불가·수요누설 위험 → 모델 피처 아님. 정제·필터·참고용.)
- **산출물: 격자 사전계산 방식.** 격자 칸마다 (예측가+오차범위+개별 SHAP top N)을 미리 계산해 테이블로 저장. 인계물 = 모델(joblib) + 격자 테이블 + 좌표→예측 함수 + README. **웹/REST 서버는 다른 팀원 몫(우리 범위 밖).**
- **SHAP 2층위**: ① 전체 경향(보고서용, 1회 고정) ② 위치별 기여(웹 `top_factors`용, 칸마다 다름).

## 환경 / 작업 규칙
- **uv 프로젝트.** 패키지 추가는 `uv add <pkg>` (pip 쓰지 말 것). 실행은 `uv run python ...`.
- **Python 3.12 고정**(`.python-version`). 시스템 python 3.14 쓰지 말 것 — 무거운 지리/ML 스택 설치가 깨질 수 있음.
- 무거운 스택: `geopandas`·`osmnx` ✅설치(도로망 단계) / `xgboost`·`shap`은 모델링 단계 도달 시 `uv add`. 현재 설치: pandas·openpyxl·xlrd·requests·python-dotenv·scipy·**osmnx·geopandas·shapely·pyproj·networkx**.
- 한국어 CSV 인코딩 주의(cp949 / utf-8-sig 혼재). 공공데이터 엑셀은 제목·조회조건 행이 위에 붙어 헤더가 밀리는 경우 있음.

## 파이프라인 (계획서 순서)
0. 가격↔좌표 매칭 ✅ → 1. 수집 ✅ → 2. 정제·정규화 ✅ → 3. spatial join ✅(좌표베이스+ARS/좌표 조인)

## 협업 스타일
- 사용자는 **ML 초심자**(전공: 컴퓨터공학/AI). 새 개념·선택지는 짧게 설명 곁들이기.
- 단계마다 데이터 분포/중간 결과를 보여주고, **미정·판단 지점에선 멈추고 묻기**(예: 분석 단위 쉘터형 vs 전체 / 격자 칸 크기 / 피처 4·6 포함 여부).
- 대량 수집(스크래핑)처럼 노이즈 큰 작업은 **별도 세션**에서. 이 디렉터리의 메인 세션은 결정·분석 위주로 유지.

## 알려진 한계 (보고서에 명시)
- 수집가는 실거래가가 아니라 **호가**일 수 있음.
- 대행사가 유동인구 기준으로 책정했다면, 유동인구로 가격 예측 = 그 책정 공식의 일부 역산일 수 있음.
- Y에 제작·설치비 포함 → 입지 무관 고정비가 섞임(대체로 모델 절편으로 흡수).
- 가격이 대행사 **단가표 계단값**(연속 아님; 하위 ~10%가 ₩205,800 한 점) → 모델은 입지→가격티어 학습 성격.
- 데이터 적은 외곽 지역 예측은 신뢰도 낮음(자치구 CV R² 0.32; `model/grid`의 `신뢰도` 열로 노출).
