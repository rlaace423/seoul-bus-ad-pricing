## 파일 목록

> 1~5번 덕플레이스 가격 파일은 모두 `data/seoul/price/` 하위. (좌표 master·`모델테이블.csv`는 `data/seoul/` 루트.)

### 1. `덕플레이스_서울_전체.csv` ⭐ 정식본 (1정류장 1행)
- **2,601개 고유 정류장**, 14컬럼, UTF-8-SIG.
- 서울 25개 구 전수 수집 → **ARS 기준 dedupe**(같은 정류장의 다중 패널 listing을 1행으로 통합).
- 좌표 마스터 매칭 **99.92%** (2,599/2,601).
- Y(광고가)의 출처. ⚠️ **영상(45개)은 `상품유형`으로 구분 — 메인 모델은 `인쇄`만 필터해서 사용.**

### 2. `덕플레이스_서울_전체_원시.csv` (원시 · 추적용)
- **2,692행** = dedupe 이전 전체 listing. 한 정류장에 광고 패널이 여러 개면 여러 행(productId 다름).
- 매칭/이상 추적·재현용. 분석은 정식본 사용.

### 3. `덕플레이스_서울_정제.csv` ⭐ 2단계 산출 (모델 입력 후보)
- 정식본 + **`면수`, `가격_원per면`** 2컬럼 추가 (16컬럼).
- `가격_원per면` = `가격_원` ÷ `면수`(단면1·양면2·전체4) → **면 기준을 통일한 정규화 Y**.
- 모델 학습: `상품유형=='인쇄'` & `가격_원per면` 존재(2,554개)를 Y로 사용.

### 3b. `덕플레이스_서울_모델용.csv` ⭐ 모델 입력 (정제본 필터 결과)
- **2,555행** = 인쇄 & 원/면 정규화 완료(영상 45 제외, 면기준없음 01118만 제외). 모델 Y = `가격_원per면`(139,650 ~ 2,234,400, 중앙 578,200).

### 4. `덕플레이스_자치구요약.csv` (참고)
- 자치구별 정류장수·평균·최저·최고가(인쇄 기준).

### 5. `덕플레이스_서울시_서초구_sample.csv` (구버전 샘플)
- 0단계 매칭 테스트용 서초 14개 수기 샘플. 전체본으로 대체됨(보존만).

### (참고) `서울시버스정류소위치정보(20260506).xlsx`
- 열린데이터광장 좌표 마스터(11,250개). 키 = `ARS_ID`. 가격↔좌표 매칭에 사용.

## 컬럼 사전 (전체본 / 정제본)
| 컬럼 | 설명 |
|---|---|
| `자치구` | 예: 서초구 |
| `정류장이름` | 사이트 표기 원문(끝 `[ARS]` 포함) |
| `ARS` | 5자리 정류장번호. **반드시 텍스트로**(leading-zero 보존, 예 `01118`) |
| `가격_원` | 월 실결제가(부가세·제작비·설치비 포함), 정직가(-2% 반영) |
| `가격기준` | `단면(1면)`/`양면(2면)`. **이 가격이 몇 면 기준인지** (정규화 핵심) |
| `가격_양면`,`가격_4면` | 미수집(공란) — Y엔 `가격_원`+`가격기준`이면 충분 |
| `상품유형` | `인쇄`/`영상`. **영상=디지털, 메인모델 제외** |
| `지원포맷` | `ai,psd`(인쇄) / `mp4,mov,wmv`(영상) |
| `운영시간`,`야간조명`,`혼잡도` | 참고용(좌표산출 불가/수요누설 → 모델 피처 아님) |
| `상세URL` | dukplace 상세 페이지 |
| `수집일시` | 2026-05-31 |
| `면수`*,`가격_원per면`* | (정제본 전용) 면 기준 파생 / 정규화 Y |

## 알려진 한계
- 가격은 **호가**일 수 있음(실거래가 아님). 제작·설치비 포함(입지 무관 고정비 섞임 → 대체로 절편 흡수).
- **미매칭 2개** (`22399` 서초포레스타입구, `03726`): 좌표 마스터에 없음 → spatial join에서 제외.
- **면기준 미표기**: `07219` 면목2동은 페이지가 "1면 기준"(파서가 "단면(1면)"만 잡고 놓침) → **단면(1면)으로 보정·포함**. `01118` KT광화문 인쇄패널은 면 기준 표기 자체가 없음(디지털 보드 주력 위치) → **모델 제외**. 결과 모델용 **2,555행**.
- **영상 45개**: 시간분할 과금·면 개념 없음 → 메인 모델 제외(`상품유형`으로 필터).
- 광고 인벤토리는 쉘터 있는 정류소라 마스터 11,250개의 부분집합(2,601개).
